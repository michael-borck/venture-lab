# Landing page screenshots

Drop app screenshots here named `venture-lab-1.png`, `venture-lab-2.png`, `venture-lab-3.png`, …
(consecutive numbers, PNG format).

The landing page (`docs/app.js`) discovers them automatically and rotates them in the
hero carousel — no code change needed. Numbering must be gap-free: discovery stops at
the first missing number. Aim for a 3:2 aspect ratio (e.g. 1200×800, the app's default
window size) so they fill the frame without cropping.

Until at least `venture-lab-1.png` exists, the page shows a styled placeholder instead.
